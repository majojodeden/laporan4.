A. Konsep Dasar PHP

1. Apa itu PHP?
PHP adalah bahasa pemrograman server-side yang digunakan untuk membangun aplikasi web dinamis.

2. Keunggulan PHP:

  • Mudah dipelajari.
  • Dukungan komunitas besar.
  • Kompatibilitas tinggi dengan banyak platform.

Contoh Hello World

<?php
echo "Hello, World!";
?>

B. Perbandingan GET dan POST

GET POST
Mengirim data melalui URL. Mengirim data melalui body HTTP.
Cocok untuk data ringan. Aman untuk data sensitif.
Data terlihat di URL. Data tidak terlihat di URL.

C. Include, Require, dan Koneksi MySQL

1. Perbedaan Include dan Require:

  • Include: Tidak fatal jika file tidak ditemukan.
  • Require: Fatal jika file tidak ditemukan.

2. Contoh Koneksi MySQL:

<?php
$conn = new mysqli("localhost", "username", "password", "database");
if ($conn->connect_error) {
    die("Koneksi gagal: " . $conn->connect_error);
}
echo "Koneksi berhasil!";
?>

D. Topik Lanjutan PHP

1. Object-Oriented Programming (OOP)

PHP mendukung pemrograman berorientasi objek, yang memungkinkan pengorganisasian kode yang lebih baik.

2. Framework PHP

Framework seperti Laravel, CodeIgniter, dan Symfony mempercepat pengembangan aplikasi web.
