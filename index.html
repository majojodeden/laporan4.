A. Konsep Dasar PHP

1. Apa itu PHP?
PHP adalah bahasa pemrograman server-side yang digunakan untuk membangun aplikasi web dinamis.

2. Keunggulan PHP:

  • Mudah dipelajari.
  • Dukungan komunitas besar.
  • Kompatibilitas tinggi dengan banyak platform.

Contoh Hello World

<?php
echo "Hello, World!";
?>

B. Perbandingan GET dan POST

GET POST
Mengirim data melalui URL. Mengirim data melalui body HTTP.
Cocok untuk data ringan. Aman untuk data sensitif.
Data terlihat di URL. Data tidak terlihat di URL.

C. Include, Require, dan Koneksi MySQL

1. Perbedaan Include dan Require:

  • Include: Tidak fatal jika file tidak ditemukan.
  • Require: Fatal jika file tidak ditemukan.

2. Contoh Koneksi MySQL:

<?php
$conn = new mysqli("localhost", "username", "password", "database");
if ($conn->connect_error) {
    die("Koneksi gagal: " . $conn->connect_error);
}
echo "Koneksi berhasil!";
?>

D. Topik Lanjutan PHP

1. Pemrograman Asinkron

PHP versi terbaru mendukung pemrograman asinkron, yang memungkinkan pengelolaan request yang lebih efisien.

2. Security dalam PHP

Mengimplementasikan teknik keamanan seperti prepared statements dan filtering input adalah penting dalam pengembangan aplikasi web dengan PHP.

E. Pemrograman Berorientasi Objek (OOP)

PHP mendukung penuh pemrograman berorientasi objek (OOP), yang menyediakan struktur kode yang lebih terorganisir dan modular.

Contoh Sederhana OOP

<?php
class Mobil {
    public $merk;
    public $warna;

    public function __construct($merk, $warna) {
        $this->merk = $merk;
        $this->warna = $warna;
    }

    public function infoMobil() {
        return "Mobil ini adalah $this->merk berwarna $this->warna.";
    }
}

$mobil1 = new Mobil("Toyota", "Merah");
echo $mobil1->infoMobil();
?>
